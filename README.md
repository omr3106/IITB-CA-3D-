# IIT Bombay Techfest 2026 Website

A polished, interactive festival website for IIT Bombay Techfest 2026 with multiple pages, a desktop-style landing experience, and route-based navigation.

## Features

- Interactive homepage with a 3D background and desktop-style layout
- Dedicated pages for Opening, Exhibitions, Workshops, Schedule, and Venue
- Express-based local routing for page navigation
- Responsive design for desktop and mobile use

## Project Structure

- index.html — main landing page
- opening.html — opening ceremony page
- exhibitions.html — exhibitions page
- workshops.html — workshops page
- schedule.html — schedule page
- venue.html — venue guide page
- style.css — shared styling
- app.js — interactive UI and visuals
- server.js — Express server for local routing

## Requirements

- Node.js
- npm

## Installation

1. Open the project folder
2. Install dependencies:

   npm install

## Run the Project

Start the local server:

npm start

Then open:

http://localhost:3000/

## Notes

- The app uses Express to serve the festival pages locally.
- If port 3000 is already in use, stop the existing process and run the server again.
